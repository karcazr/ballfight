<Ball Fight 규칙>
1. 이 게임의 공에 해당하는 엔티티(캐릭터)는 편의상 사람이라고 부르기로 함.
2. 모든 사람은 기본 스탯이 정해져있음. (체력, 보호막, 이동 속도)
    2-1. 일반적으로 사람은 체력이 모두 닳는다면 패배함. 일반적으로 체력은 자동으로 차지 않고 스킬이나 궁극기를 통해서만 회복됨.
    2-2. 보호막이란 사람이 받는 피해를 흡수하는 요소임. 보호막은 체력과 독립되어 있으므로, 체력 회복 스킬의 영향을 받지 않음. 보호막이 닳는 것은 피해를 입은 판정이 아님.
    2-3. 보호막은 피해를 먼저 흡수하고, 보호막이 부족할 경우 남은 피해만 체력에 들어감. 예: 보호막 50 상태에서 60 피해를 받으면 보호막 50이 사라지고 체력 10이 감소함.
    2-4. 보호막이 0이 되는 순간 보호막 파괴 사운드가 재생됨.
    2-5. 이동 속도는 사람이 일반적으로 이동하는 속도임. 단위는 px/s.
3. 모든 사람은 패시브 또는 액티브 스킬이 있음. 모든 액티브 스킬은 쿨타임이 돌고, 이는 공 주변에 돌아가는 하얀색 테두리로 표시함.
4. 모든 사람은 궁극기 라는 강력한 스킬이 있음. 궁극기 충전 게이지를 100%까지 모으면 자동으로 사용함. 단, 캐릭터별 시전 시간이나 기존 스킬 종료 대기 규칙이 우선됨.
    4-1. 궁극기 게이지는 보호막과 체력에 실제로 입힌 총 피해에 비례해 참. 일반적으로 총 200의 피해를 입혀야 궁극기가 참. 즉 1의 피해를 입힐 때마다 궁극기가 0.5%씩 참.
    4-2. 궁극기 게이지는 실제 체력 회복량에 비례해 참. 일반적으로 총 200의 체력을 회복해야 궁극기가 참. 즉 1의 체력을 회복할 때마다 궁극기가 0.5%씩 참.
    4-3. 보호막에 들어간 피해도 궁극기 게이지를 채움. 무적으로 인해 피해가 0이면 게이지가 차지 않음.
    4-4. 궁극기 시전 중에는 궁극기 게이지가 차지 않고, 궁극기로 인한 회복이나 피해에도 게이지가 차지 않음.
5. 모든 사람은 벽에 부딪히면 튕기고, 공격을 받았을 때도 튕겨나감.
    5-1. 체력이 0이 되면 즉시 이동, 스킬, 충돌 판정이 중단되고 파괴 이펙트와 함께 경기장에서 사라짐.
6. 스킬이 아니라면, 모든 사람은 접촉 시에 피해를 입히지 않음. 튕겨 나가는 판정은 벽과 동일한 판정임.
7. 궁극기 시전 도중에 일반적으로 기본 스킬은 사용되지 않음.
8. 상태이상의 종류(기절, 취약, 약화, 속박, 둔화, 수면)
    8-1. 기절은 이동 불가, 스킬 사용 불가이고, 스킬 쿨타임이 돌지 않으며, 이지호 오라 같은 패시브 스킬 또한 잠시 꺼진다.
    8-2. 취약은 받는 피해를 50% 증가시킨다.
    8-3. 약화는 주는 피해를 33% 감소시킨다.
    8-4. 속박은 이동 불가만 된다. 공격이나 스킬 사용은 가능하다.
    8-5. 둔화는 이동 속도를 일정 비율만큼 감소시킨다. 현재 구현은 50% 둔화.
    8-6. 수면은 기절과 같지만, 체력 피해를 입는다면 즉시 풀린다.
9. 상태이상은 공 윗부분에 한글로 표시를 한다. 항상 느낌표를 붙이고 눈에 잘 띄게 한다. (예. 기절!, 취약!, 약화!)
10. 배속 버튼은 1배, 2배, 4배가 있으며 이동, 쿨타임, 피해/회복 틱, 투사체 등 시뮬레이션 전체에 적용됨.
11. 전투 중에는 중단/재생 버튼, 다시하기 버튼, 선택으로 버튼이 있음.
    11-1. 중단 상태에서는 중단 버튼이 재생 버튼으로 바뀌고, 누르면 이어서 진행함.
    11-2. 다시하기는 같은 캐릭터 조합으로 즉시 새 경기를 시작함.
    11-3. 홈 화면에서 시작을 누르면 1vs1 또는 1vs1vs1 모드를 선택함. 1vs1vs1은 세 캐릭터가 동시에 전투하며 마지막 생존자 한 명이 승리함.
12. 효과음은 외부 파일이 아니라 Web Audio 합성으로 생성함.
    12-1. 첫 사용자 클릭 이후 오디오가 활성화됨.
    12-1-1. 브라우저 자동재생 제한이나 탭 전환으로 Web Audio가 중단되면 다음 포인터/키 입력에서 다시 활성화함. Web Audio가 계속 잠겨 있으면 생성된 WAV를 사용하는 HTML Audio 대체 출력으로 효과음을 재생함.
    12-2. 벽 충돌, 공 충돌, 피해, 보호막 파괴, 김성윤 스킬/궁극기, 이지호 오라/궁극기, 배정대 파동/궁극기, 이서준 글자/궁극기, 기절에 각각 다른 소리를 사용함.

Skill: 5초마다 무작위 상대를 지정하고 0.3초 동안 멈춰 대상과 광선으로 연결된 뒤 20의 피해를 입히고 1원을 얻음. 1원 이상은 이동속도 100 증가, 3원은 공격력 10 증가, 5원 도달 시 체력 100 회복, 7원은 스킬 쿨타임 1초 감소, 10원은 공격력 10 추가 증가, 15원은 생명력 흡수 50%와 최대 체력 50 증가, 20원은 스킬 쿨타임 1초 추가 감소, 30원은 공격력 100 추가 증가. 게임당 한 번 10원 이상 보유한 상태로 죽으면 10원을 소모하고 최대 체력으로 부활함.
Ult: 시전 위치를 중심으로 반경 500px의 보라색 영역을 15초 동안 전개함. 영역 안의 적은 이동속도가 80% 감소하고 약화되며 어떠한 방법으로도 체력을 회복할 수 없음.
Note for codex: 일반 스킬은 리그 오브 레전드 세나의 평타처럼 박예찬과 피해자를 보라색/금색 에너지 광선으로 연결하고, 대상에게서 박예찬 쪽으로 돈 입자가 빨려 들어오는 모션을 사용함. 보유 금액이 줄면 현재 금액보다 높은 단계의 버프도 제거됨. 생명력 흡수는 실제 입힌 피해의 50%를 회복함. 궁극기는 지속 시전 상태가 아니라 사용 즉시 시전 위치에 영역을 설치하고 끝나는 방식이며, 영역이 유지되는 15초 동안에도 박예찬은 이동과 일반 스킬 사용이 가능함. 궁극기 영역이 유지되는 동안에는 궁극기 게이지가 충전되지 않음. 빼앗긴 돈은 상대 머리 위에 누적 횟수로 표시하지만 상대의 별도 능력치를 감소시키지는 않음. 박예찬 HUD에는 보유 금액과 현재 돈으로 활성화된 이동속도, 공격력, 5원 회복 보상, 쿨타임 감소, 생명력 흡수, 최대 체력, 부활 가능 효과를 현재 적용 수치로 표시함. 궁극기 영역은 시전 위치에 고정되며 웅장한 사운드를 사용함. 캐릭터 이미지는 pyc.png를 사용함.


# Codex 추천 주석:
# - 버전명에 alpha 또는 beta가 포함된 커밋은 개발 중간 버전이므로 patchnote.txt에 기록하지 않음. 정식 출시 버전만 패치노트에 기록함.
# - 모든 캐릭터는 어떠한 방식으로든 부활하는 순간 박예찬에게 빼앗겨 누적된 잃은 돈 스택이 0으로 초기화됨.
# - 궁극기 게이지의 "입힌 피해"는 보호막 피해와 체력 피해를 합친 실제 총 피해로 계산함.
# - "체력 회복"으로 궁극기 게이지가 찰 때도 실제로 회복된 양만 계산하면 최대 체력 상태에서 게이지가 차는 문제를 막을 수 있음.
# - 기본 접촉은 피해 없이 튕김만 발생하고, 접촉 피해는 김성윤 강화 공격/궁극기 공격처럼 스킬 효과가 켜져 있을 때만 발생한다고 명시하면 좋음.
# - 궁극기는 게이지가 100%가 되는 즉시 자동 시전되는지, 플레이어가 직접 누르는 액티브인지 계속 명시해두면 좋음. 현재 구현은 자동 시전 기준.
# - 이지호 궁극기는 보호막이 일부 남아 있으면 유지되고, 보호막이 완전히 파괴되거나 체력이 실제로 감소하면 해제됨.
# - 사운드는 현재 합성음이므로, 외부 효과음으로 교체할 경우 각 파일 라이선스와 출처 표기 규칙을 별도 관리하는 것이 좋음.
