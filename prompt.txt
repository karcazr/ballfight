<Ball Fight 규칙>
1. 이 게임의 공에 해당하는 엔티티(캐릭터)는 편의상 사람이라고 부르기로 함.
2. 모든 사람은 기본 스탯이 정해져있음. (체력, 보호막, 이동 속도)
    2-1. 일반적으로 사람은 체력이 모두 닳는다면 패배함. 일반적으로 체력은 자동으로 차지 않고 스킬이나 궁극기를 통해서만 회복됨.
    2-2. 보호막이란 사람이 받는 피해를 흡수하는 요소임. 보호막은 체력과 독립되어 있으므로, 체력 회복 스킬의 영향을 받지 않음. 보호막이 닳는 것은 피해를 입은 판정이 아님.
    2-3. 보호막은 피해를 먼저 흡수하고, 보호막이 부족할 경우 남은 피해만 체력에 들어감. 예: 보호막 50 상태에서 60 피해를 받으면 보호막 50이 사라지고 체력 10이 감소함.
    2-4. 보호막이 0이 되는 순간 보호막 파괴 사운드가 재생됨.
    2-5. 이동 속도는 사람이 일반적으로 이동하는 속도임. 단위는 px/s.
3. 모든 사람은 패시브 또는 액티브 스킬이 있음. 모든 액티브 스킬은 쿨타임이 돌고, 이는 공 주변에 돌아가는 하얀색 테두리로 표시함.
4. 모든 사람은 궁극기 라는 강력한 스킬이 있음. 궁극기 충전 게이지를 100%까지 모으면 자동으로 즉시 사용함.
    4-1. 궁극기 게이지는 실제 체력 피해에 비례해 참. 일반적으로 총 200의 체력 피해를 입혀야 궁극기가 참. 즉 1의 체력 피해를 입힐 때마다 궁극기가 0.5%씩 참.
    4-2. 궁극기 게이지는 실제 체력 회복량에 비례해 참. 일반적으로 총 200의 체력을 회복해야 궁극기가 참. 즉 1의 체력을 회복할 때마다 궁극기가 0.5%씩 참.
    4-3. 보호막에만 들어간 피해는 궁극기 게이지를 채우지 않음.
    4-4. 궁극기 시전 중에는 궁극기 게이지가 차지 않고, 궁극기로 인한 회복이나 피해에도 게이지가 차지 않음.
5. 모든 사람은 벽에 부딪히면 튕기고, 공격을 받았을 때도 튕겨나감.
6. 스킬이 아니라면, 모든 사람은 접촉 시에 피해를 입히지 않음. 튕겨 나가는 판정은 벽과 동일한 판정임.
7. 궁극기 시전 도중에 일반적으로 기본 스킬은 사용되지 않음.
8. 상태이상의 종류(기절, 취약, 약화, 속박, 둔화, 수면)
    8-1. 기절은 이동 불가, 스킬 사용 불가이고, 스킬 쿨타임이 돌지 않으며, 패시브 스킬 또한 잠시 꺼진다.
    8-2. 취약은 받는 피해를 50% 증가시킨다.
    8-3. 약화는 주는 피해를 33% 감소시킨다.
    8-4. 속박은 이동 불가만 된다. 공격이나 스킬 사용은 가능하다.
    8-5. 둔화는 이동 속도를 일정 비율만큼 감소시킨다. 현재 구현은 50% 둔화.
    8-6. 수면은 기절과 같지만, 체력 피해를 입는다면 즉시 풀린다.
9. 상태이상은 공 윗부분에 한글로 표시를 한다. 항상 느낌표를 붙이고 눈에 잘 띄게 한다. (예. 기절!, 취약!, 약화!)
10. 배속 버튼은 1배, 2배, 4배가 있으며 이동, 쿨타임, 피해/회복 틱, 투사체 등 시뮬레이션 전체에 적용됨.
11. 전투 중에는 중단/재생 버튼, 다시하기 버튼, 선택으로 버튼이 있음.
    11-1. 중단 상태에서는 중단 버튼이 재생 버튼으로 바뀌고, 누르면 이어서 진행함.
    11-2. 다시하기는 같은 캐릭터 조합으로 즉시 새 경기를 시작함.
12. 효과음은 외부 파일이 아니라 Web Audio 합성으로 생성함.
    12-1. 첫 사용자 클릭 이후 오디오가 활성화됨.
    12-2. 벽 충돌, 공 충돌, 피해, 보호막 파괴, 김성윤 스킬/궁극기, 이지호 오라/궁극기, 배정대 파동/궁극기, 이서준 글자/궁극기, 기절에 각각 다른 소리를 사용함.

# Codex 추천 주석:
# - "입힌 피해"는 보호막 피해가 아니라 실제 체력이 줄어든 양으로 계산하는 편이 보호막 규칙과 충돌이 적음.
# - "체력 회복"으로 궁극기 게이지가 찰 때도 실제로 회복된 양만 계산하면 최대 체력 상태에서 게이지가 차는 문제를 막을 수 있음.
# - 기본 접촉은 피해 없이 튕김만 발생하고, 접촉 피해는 김성윤 강화 공격/궁극기 공격처럼 스킬 효과가 켜져 있을 때만 발생한다고 명시하면 좋음.
# - 궁극기는 게이지가 100%가 되는 즉시 자동 시전되는지, 플레이어가 직접 누르는 액티브인지 계속 명시해두면 좋음. 현재 구현은 자동 시전 기준.
# - 보호막이 깎이는 것은 피해를 입은 판정이 아니므로, 이지호 궁극기는 체력이 실제로 감소할 때만 해제된다고 명시하면 좋음.
# - 사운드는 현재 합성음이므로, 외부 효과음으로 교체할 경우 각 파일 라이선스와 출처 표기 규칙을 별도 관리하는 것이 좋음.
